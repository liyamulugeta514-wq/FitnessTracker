# Fitness Tracker

A Kotlin Object-Oriented Programming project that helps users track fitness information and receive personalized exercise recommendations.

## Features

- User Profiles
- BMI Calculation
- BMI Category Classification
- Water Intake Recommendation
- Personalized Exercise Suggestions
- Workout Summaries
- Calorie Estimation

## OOP Concepts Used

- Classes and Objects
- Encapsulation
- Inheritance
- Polymorphism
- Collections (mutableListOf)
